
# INSTALLFIRE - Instale e atualize o Firefox no seu GNU/Linux, de forma rápida e fácil.

Para conferir o modo de uso:

	./installfire.sh -h

Para configurar o Programa no Sistema, use:

	./installfire.sh -c
    
Configurado, chamada simples com [ instfire ].
